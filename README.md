# Web Development Project 5 - CooperHewittData

Submitted by: Jessica Liao

This web app: Displays data for Contemporary Art that is currently on display at Cooper Hewitt Museum. It contains creation year statistics and lists all pieces in a table with creation year, art medium and title

Time spent: 5 hours spent in total
